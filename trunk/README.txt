
win-context-menu is a shell extension that allows the user to configure his custom menu items and execute a certain program upon clicking on these menu items.

Installation: 
Run install.bat.

Uninstallation:
Run uninstall.bat.

If explorermenu.xml doesn't exist in the directory of win_context_menu.dll, it will be created automatically. Read the comments in explorermenu.xml for instructions about how to use this tool.
